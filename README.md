# Tetris

This is a game written in C++.


## How to run

Open up a terminal and enter:

```
# Clone the directory first:
git clone https://github.com/RobbieZhao/Tetris.git

# Change your directory:
cd build

# Create a makefile
cmake ..

# Build project
make

# Run
./Tetris
```
